# 귤 주문 웹페이지 요구사항

## 프로젝트 개요
제주 귤을 온라인으로 주문할 수 있는 간단한 웹페이지

## 페이지 구성

### 1. 메인 페이지 (귤 상세정보 페이지)
- 귤 상품 이미지
- 상품명 및 가격 정보
- 귤 상세 설명 (원산지, 당도, 특징 등)
- 주문 페이지로 이동하는 버튼

### 2. 주문 페이지
- **구매자 정보 입력**
  - 이름 (필수)
  - 전화번호 (필수)
  - 배송 주소 (필수)

- **주문 정보 입력**
  - 귤 수량 선택 (박스 단위)
  - 총 결제 금액 자동 계산

- **주문하기 버튼**
  - 입력값 유효성 검사
  - 주문 완료 알림

## 추가 기능
- 반응형 디자인 (모바일 지원)
- 수량 변경 시 가격 자동 계산
- 입력 폼 유효성 검사
- 네비게이션 바 (페이지 간 이동)

## 기술 스택
- HTML5
- CSS3
- JavaScript (Vanilla)

## 파일 구조
